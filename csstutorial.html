Anyhow, I'll let you in on a little trick I've learned for setting the site color quickly without too much typing . Simply just look at the below code! 
  
    <style>
      body {
      background-color: #69779e;
      text-size-adjust: 20;
      text-align: initial;
    }
    </style>

See, that wasn't that hard now was it?

To change the font type you do something similar, but not quite the same. You can change the font family using the <font> tag. See below for another easy example! I simply linked to an external web font, so it's a bit more complicated than changing the font within a text editing program, but not too terribly hard at all! Not even for a beginner!

  
    <style>
    p style{
      @font-face {
     font-family:XfinityStandard;
     src:url(https://static.cimcontent.net/common-web-assets/fonts/xfinity-standard-optimized/xfinitystandard-exlt.woff2) format("woff2"),
     url(https://static.cimcontent.net/common-web-assets/fonts/xfinity-standard-optimized/xfinitystandard-exlt.woff) format("woff");
     font-weight:200;
     font-style:normal
    }
    }
    
OR
    body {
      font-family: "Lato", sans-serif;
    }
    </style>

I have found both are equally effective ways to change your font, but that linking to a external font is a bit more exciting, because you're doing more than what the average person would do. Most people I know keep web fonts at basic ones, (Unless your a scammer that uses comic sans.
